The Idea

Opening a restaurant can be a huge risk. There are so many factors to consider like menu, location, staffing, and pricing. What if there was a way to predict if a restaurant would be successful?

Our analysis takes the information from restaurants in the San Francisco area to create a model to predict success. This model uses ratings, price, location, and cuisines from the restaurants to build the model.

This model is using San Francisco data currently, but it is designed to use any city, state, or country's data in the future.
